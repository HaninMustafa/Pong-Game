# Pong Game

A basic pong game using HTML5 Canvas, ES2015, SASS and Webpack build tool.

## Setup

##### Install

`> npm i`

##### Run

`> npm start`

## Keys

##### Player 1:
* a: up
* z: down
* q: fire a new ball

##### Player 2:
* ▲ : up
* ▼: down
* /: fire a new ball
